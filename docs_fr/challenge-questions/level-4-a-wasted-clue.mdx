- Tous les 3 sont joués sur les stacks.
- Alice clue numéro 3 à Cathy, touchant une carte en slot 3.
- Quel type de clue est-ce ?
- Quelles notes doivent être écrites ?

<Question />

</TabItem>
<TabItem value="solution">

- Aucun 3 supplémentaire ne devrait être joué sur les stacks, donc la carte touchée est du trash connu.
- C'est un _Trash Chop Move_, qui _Chop Move_ toutes les cartes à droite du trash.
- Tous les joueurs devraient écrire `kt` sur le slot 3 de Bob et `cm` sur les slots 4 et 5 de Bob.
