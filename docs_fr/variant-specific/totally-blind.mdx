Ces conventions s'appliquent aux variantes "Totally Blind".

### Clue Interpretation Tables

- De manière similaire aux conventions des variantes _Color Blind_, nous définissons des interprétations arbitraires pour la signification de chaque clue.

#### 2-Player & 3-Player

| Clue     | Interprétation                                                       |
| -------- | -------------------------------------------------------------------- |
| Number 1 | Jouer slot 1                                                         |
| Number 2 | Jouer slot 2                                                         |
| Number 3 | Jouer slot 3                                                         |
| Number 4 | Jouer slot 4                                                         |
| Number 5 | Jouer slot 5                                                         |
| Red      | _Chop Move_                                                          |
| Yellow   | Jouer slot 1 + jouer slot 2                                          |
| Green    | Jouer slot 1 + jouer slot 3                                          |
| Blue     | Jouer slot 1 + jouer slot 4                                          |
| Purple   | Jouer slot 1 + jouer slot 5                                          |
| Teal     | Double _Chop Move_ (ou un clue nul si en situation de stall)         |

#### 4-Player & 5-Player

| Clue     | Interprétation                                                                              |
| -------- | ------------------------------------------------------------------------------------------- |
| Number 1 | Jouer slot 1                                                                                |
| Number 2 | Jouer slot 2                                                                                |
| Number 3 | Jouer slot 3                                                                                |
| Number 4 | Jouer slot 4                                                                                |
| Number 5 | _Chop Move_                                                                                 |
| Red      | Double _Chop Move_ (ou un clue nul si en situation de stall)                                |
| Yellow   | Jouer slot 1 + jouer slot 2                                                                 |
| Green    | Jouer slot 1 + jouer slot 3                                                                 |
| Blue     | Jouer slot 1 + jouer slot 4                                                                 |
| Purple   | Deux joueurs jouent leur _Finesse Position_ (cluer la personne avec le 2e blind-play)       |
| Teal     | Trois joueurs jouent leur _Finesse Position_ (cluer la personne avec le 3e blind-play)      |

#### 6-Player

| Clue     | Interprétation                                                                                          |
| -------- | ------------------------------------------------------------------------------------------------------- |
| Number 1 | Jouer slot 1                                                                                            |
| Number 2 | Jouer slot 2                                                                                            |
| Number 3 | Jouer slot 3                                                                                            |
| Number 4 | _Chop Move_                                                                                             |
| Number 5 | Double _Chop Move_ (ou un clue nul si en situation de stall)                                            |
| Red      | Deux joueurs jouent leur _Second Finesse Position_ (cluer la personne avec le 2e blind-play)            |
| Yellow   | Jouer slot 1 + jouer slot 2                                                                             |
| Green    | Jouer slot 1 + jouer slot 3                                                                             |
| Blue     | Jouer slot 1 + jouer slot 2 + jouer slot 3                                                              |
| Purple   | Deux joueurs jouent leur _Finesse Position_ (cluer la personne avec le 2e blind-play)                   |
| Teal     | Trois joueurs jouent leur _Finesse Position_ (cluer la personne avec le 3e blind-play)                  |

### Advanced Conventions

- Dans les variantes Totally Blind, les _Ejections_ et _Discharges_ fonctionnent assez différemment. Voir les [Conventions spécifiques aux variantes No Positive Clues](no-positive-clues.mdx).
