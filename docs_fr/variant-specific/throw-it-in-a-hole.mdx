title: Throw It in a Hole
---

Ces conventions s'appliquent aux variantes "Throw It in a Hole".

### Hard Variant

- Throw It in a Hole est considéré comme une [_Hard Variant_](../variant-specific.mdx#hard-variants--easy-variants), ce qui signifie que les [_Loaded Play Clues_](../variant-specific.mdx#the-loaded-play-clue-in-hard-variants-lpc) sont "activés", etc.

### No Bluffs

- Les _Bluffs_ sont désactivés.
- Il est correct d'effectuer un _Trash Finesse_ pour la dernière carte d'un rang ou d'une suite particulière, mais les _Trash Bluffs_ sont désactivés.

### Priority Changes

- Comme les 5 ne donnent pas de clue de retour, ils n'ont pas de _Priority_ sur les autres cartes de la même manière qu'à la normale.
