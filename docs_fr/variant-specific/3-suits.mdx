title: 3 Suits
---

Ces conventions s'appliquent à toute variante n'ayant que 3 suites.

### No 3 Bluffs

- La convention _3 Bluff_ est "désactivée". (Cela vient du fait que les _Double Finesses_ sont beaucoup plus probables dans ces variantes.)

### The Full Finesse

- Les conventions suivantes sont "désactivées" :
  - _5 Stall_
  - _5's Chop Move_
  - _5 Pull_
  - _5 Color Ejection_
  - _5 Number Ejection_
  - _5 Number Discharge_
- Si un 5 est cluéé, alors il est traité comme une _Finesse_ sur la suite complète de 1 à 5.
