Ces conventions s'appliquent à toute variante avec une suite dark prism.

### Prism Conventions

- Dark prism "hérite" de toutes les [conventions pour les cartes black](black.mdx) et de toutes les [conventions pour les cartes prism](prism.mdx).
- Quelques précisions et exceptions spécifiques sont listées ci-dessous.

### Dark Prism Saves

- Les joueurs doivent sauver les dark prism 3 et dark prism 4 avec un clue de nombre.
- Il y a deux exceptions à cela :
  - si le clue de couleur a aussi touché une autre carte critique visible dans le trash (par exemple pas un 5 et pas dark prism)
  - si la couleur correspond à une suite déjà entièrement jouée ou comptabilisée
- De plus, selon l'_Always Loaded Principle_, les _Save Clues_ avec number 3 ou number 4 sont toujours _Loaded_.
