<TabItem value="question">

- Le green 3 doit être joué sur les stacks.
- Quelle action Alice doit-elle effectuer ?
- Si Alice doit donner une clue, quelle card note sera écrite sur la carte focalisée ?

<GoodTouchPrincipleQuestion4 />

</TabItem>
<TabItem value="solution">

- Une clue de couleur comme une clue de chiffre dupliqueraient le green 3 et violeraient le _[Good Touch Principle](./good-touch-principle.mdx)_.
- Ainsi, Alice doit attendre patiemment que Bob défausse le green 3. Une fois la première copie du green 3 défaussée, l'équipe pourra cluer le green 3 restant.
- Puisqu'il n'y a pas de clues légales à donner, Alice doit défausser.
