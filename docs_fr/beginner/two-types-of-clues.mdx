- Fondamentalement, Hanabi consiste à donner des clues. Différents groupes ont différentes définitions de ce que les clues doivent signifier.
- Dans notre groupe, les clues ont **toujours** une signification précise.
- Les joueurs ont **uniquement** le droit de donner **deux types de clues** :
  - une _[Play Clue](./play-clues.mdx)_ (signifie de jouer la carte focalisée)
  - une _[Save Clue](./save-clues.mdx)_ (signifie de sauvegarder la carte focalisée pour plus tard)
- Si les joueurs ne peuvent pas donner de _Play Clue_ ou de _Save Clue_ valide, alors ils **doivent défausser**.
- Voyons les _Play Clues_ et les _Save Clues_ individuellement.
