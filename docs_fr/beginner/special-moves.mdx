- Hanabi est un jeu où l'on essaie d'être aussi efficace que possible, puisqu'il y a un nombre limité de clues.
- Ainsi, l'idéal est de faire jouer **deux** cartes ou plus à partir d'**une seule** clue.
- Les joueurs peuvent utiliser deux « coups » spéciaux pour cela : les _[Prompts](./prompt.mdx)_ et les _[Finesses](./finesse.mdx)_.
- Voyons chacun individuellement.
