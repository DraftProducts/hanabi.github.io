- Jusqu'à présent, nous avons couvert :
  - comment défausser (i.e. la [chop](./chop.mdx))
  - comment savoir ce qu'une clue signifie (i.e. une _[Play Clue](./play-clues.mdx)_ ou une _[Save Clue](./save-clues.mdx)_)
  - comment déterminer le focus d'une clue (i.e. _Chop-Focus_ ou plus à gauche)
- Mais cela ne vous dit pas grand-chose sur la _stratégie_ impliquée à Hanabi. Comment les joueurs trouvent-ils le meilleur coup pour une situation donnée ?
- Hanabi est très complexe, il est donc impossible d'écrire un guide sur la façon de trouver le meilleur coup. (Et c'est une bonne chose, car le jeu reste intéressant à jouer, même après des milliers de parties !)
- Heureusement, les débutants peuvent se concentrer sur l'intériorisation de seulement **trois principes principaux** :
  1. _[Good Touch Principle](./good-touch-principle.mdx)_
  1. _[Save Principle](./save-principle.mdx)_
  1. _[Minimum Clue Value Principle](./minimum-clue-value-principle.mdx)_
- Voyons chaque principe individuellement.
