- Quelle action Alice doit-elle effectuer ?

<GoodTouchPrincipleQuestion1 />

<TabItem value="solution">

- Alice doit donner une _[Play Clue](./play-clues.mdx)_ au blue 3 de Bob avant qu'il ne soit défaussé.
- Si Alice clue blue, cela focaliserait le blue 3, mais le blue 2 serait dupliqué (parce qu'il est déjà joué sur les stacks).
- Si Alice clue number 3, cela focaliserait le blue 3, mais cela toucherait aussi le red 3. Le red 3 sera utile à l'équipe dans le futur, donc c'est une bonne chose.
- Ainsi, Alice doit cluer number 3 à Bob.
