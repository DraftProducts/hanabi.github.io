<TabItem value="question">

- Alice a un 2 clué dans sa main d'une couleur inconnue. (Il a été _[2 Saved](./2-save.mdx)_ plus tôt dans la partie.)
- Quelle action Alice doit-elle effectuer ?
- Si Alice doit donner une clue, quelle card note sera écrite sur la carte focalisée ?

<GoodTouchPrincipleQuestion3 />

</TabItem>
<TabItem value="solution">

- Alice pourrait cluer yellow à Bob ou cluer number 2 à Bob. Les deux clues seraient une _[Play Clue](./play-clues.mdx)_ sur le yellow 2.
- Cependant, il est possible que le 2 clué d'Alice soit un yellow 2. Si c'est le cas, alors Alice violerait le _[Good Touch Principle](./good-touch-principle.mdx)_ et gaspillerait une clue.
- Ainsi, ce n'est pas la responsabilité d'Alice de cluer le yellow 2. Quelqu'un d'autre dans l'équipe s'en occupera. Alice doit défausser, puisqu'il n'y a pas d'autres clues à donner.
